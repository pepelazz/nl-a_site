Создание и редактирование тестовых вопросов в сервисе EduQuiz

Существует два способа создания тестовых вопросов и заданий на сервисе EduQuiz:

  1. Загрузка вопросов и заданий из электронной таблицы (файл MS Excel)

  2. Создание вопросов и заданий через панель администратора на сервисе EduQuiz



Способ #1: Загрузка вопросов и заданий из электронной таблиц

Данный способ позволяет создавать тесты (задания) за считанные секунды



Видео-инструкция о том как создавать тесты на сервисе EduQuiz при помощи электронной таблицы (MS Excel)


Скачайте пример файла с вопросами, размеченными в электронной таблице (MS Excel)


Один вопрос - одна строка электронной таблицы: ключевая логика создания тестовых вопросов в сервисе EduQuiz

Атрибуты вопроса:

  1. id вопроса - идентификационный номер вопроса

    Должен быть уникальным в рамках одной сессии одного преподавателя

    Для простоты использования рекомендуем указывать в качестве id вопроса его порядковый номер в таблице (как на примере выше)
  2. Текст вопроса - текст вопроса, который задается обучающемуся

    В случае если вопрос задается относительно иллюстрации (флаг, формула и т.п.), то ссылка на иллюстрацию указывается отдельно в поле фото
  3. Тип вопроса - тип вопроса (один верный вариант, несколько верных вариантов, открытый вопрос)

    one_choice - данное значение прописывается в ячейке типа вопроса для вопросов с одним вариантом верного ответа

    multi_choice - данное значение прописывается в ячейке типа вопроса для вопросов с несколькими вариантами верных ответов

    write_answer - данное значение прописывается в ячейке типа вопроса для открытого типа вопросов - ответ вводится с клавиатуры учеником самостоятельно

    ВАЖНО: для повышения эффективности использования открытых вопросов существует функция “Варианты для открытых вопросов” (см. пункт 9)
  4. Правильный(е) ответ(ы) - вариант правильного ответа

    Несколько правильных вариантов ответов (тип вопроса multi_choice) указываются в одной ячейке через точку с запятой и один пробел - Москва; Новосибирск; Барнаул
  5. Неправильные варианты - варианты некорректных ответов

    Несколько неправильных вариантов ответов (тип вопросов one_choice, multi_choice) указываются в одной ячейке через точку с запятой и один пробел - Берлин; Осло; Прага

    Не указывается для открытого типа вопроса (write_answer). Ячейка остается пустой
  6. Тэг (тема) вопроса - тэг вопроса, по которому он относится к конкретному заданию

    Механика ТЭГов используется для управления заданиями (тестами)

    За счет тэга(ов) происходит установление соответствия в паре “Тест (задание) - вопрос(ы)”

    Каждое задание (тест) также содержит ТЭГ

    При загрузке файла с вопросами сервис EduQuiz создает задания (тесты) на основе логики тэгов. В каждый тест выбираются (включатся) вопросы с совпадающими тэгом(ами)

    В приведенном выше примере при загрузке файла на сервис EduQuiz будут созданы 3 теста (задания): “Про одежду”, “Про цвета”, “Про космос”

    В каждое из заданий попадут вопросы с указанными тэгами “работа”; “фрукты” + “овощи”; “космос”

    Использование данной механики управления банком вопросов приобретает ценность при управлении значительным объемом вопросов (100+) на смежные тематики

    В случае использования незначительного количества вопросов и заданий рекомендуется указывать простые тэги, например - тему занятия (“простые_дроби”)
  7. Баллы за вопрос - количество баллов, которое получает ученик в случае верного ответа

    Для вариантов с несколькими вариантами правильных ответов (тип multi_choice): в случае, если ученик указал лишь некоторые верные ответы (но не все), либо же указал в т.ч. и неверные ответы - он получает ноль баллов
  8. Фото вопроса - картинка (иллюстрация) к вопросу

    В ячейке прописывается ссылка на картинку, размещенную в интернете. Вид ссылки: http://nl-a.ru/img/header-logo.png

    Иллюстрации могут быть найдены либо через поисковики (яндекс, google), либо загружены самостоятельны на открытые хостинги фотографий (например: альбомы в vk.com)

    Данный атрибут является опциональным. Если иллюстрации нет - ячейка остается пустой
  9. Варианты regex для открытых вопросов - синонимы (корни слов) варианта верного ответа

    Типовая проблема автоматизированной проверки открытых вопросов: обучающиеся часто пишут ответ, который не строго соответствует прописанному правильному ответу в системе, но строго совпадает по смыслу

    Например вместо правильного ответа “Юрий” тестируемый может написать “Юра”. Смысл (семантика) совпадает; синтаксис - нет

    Для минимизации подобных рисков при автоматизированном использовании открытых вопросов в сервисе EduQuiz предсмотрен функционал “создания синонимов” (регулярные выражения, regex) правильных ответов

    В данной ячейке прописываются “синонимы” корректного ответа

    Рекомендуем прописывать лишь корни ответа с указанием специального символа * в конце корня. Несколько вариантов ответов указываются через точку с запятой через пробел
    Пример использования “Регулярного выражения”:
    Вопрос: “Укажите одну из исторических столицу России”

    Верный ответ: Москва

    Варианты ответов: Москв* ; Ладог* ; Новгород* ; Киев* ; Владимир* ; Санкт* ; Петроград*

    Между “звездочкой” и точкой с запятой не должно быть пробела в ячейке; в примере стоит пробел в силу особенностей форматирования текста на веб-сайтах





Способ #2: Создание вопросов и заданий через панель администратора на сервисе EduQuiz

Для создания вопроса непосредственно на сервисе EduQuiz необходимо авторизовать (через профиль в одной из социальных сетей)


Затем нажать на кнопку “Create” (создать)



Затем в меню (слева вверху) нажать на вкладку “Вопросы”





В разделе управления вопросами для создания нового вопроса - нажать кнопку “Плюс” (справа внизу)





Заполнить все поля вопроса


Нажать кнопку “Сохранить”

Все. Вопрос создан и сохранен на сервисе EduQuiz

ВАЖНО: сервис EduQuiz не хранит вопросы длительное время. Вопросы удаляются через 23 часа после создания, либо в момент окончания сеанса тестирования

Для сохранности создаваемого контента рекомендуется сохранять (выкачивать) с сервиса файлы с вопросами. Для этого во вкладке “Вопросы” необходимо нажать на иконку “скачать файл с вопросами”





На диск вашего устройства сохранится файл в формате MS Excel с размеченными вопросами и тестами (заданиями)

Разместите файл в удобной для вас папке на вашем устройстве (так чтобы вы могли впоследствии его легко найти и быстро создать новые тесты)
